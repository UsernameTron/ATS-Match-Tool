# ATS Resume Optimizer

AI-powered resume optimization tool that automatically enhances resumes to improve ATS match scores for Customer Success roles.

## Features
- Real-time ATS score analysis
- Automatic resume optimization
- Keyword density optimization
- Domain alignment checking
- Category-specific scoring

## Setup
1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Add OpenAI API key to `.env`
4. Run: `streamlit run app.py`
